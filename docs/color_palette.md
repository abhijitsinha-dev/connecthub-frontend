# Color Palette

## ☀️ Light Mode
The light theme relies on high-contrast dark text on bright white surfaces, mimicking a clean, classic app interface.

- **Primary Background**: `#FFFFFF` (Pure White) — Used for the main feed and content areas.
- **Secondary Background**: `#F3F4F6` (Cool Light Gray) — Used for sidebars, navigation bars, or borders separating posts.
- **Primary Accent / Brand**: `#4F46E5` (Vibrant Indigo) — Used for primary actions like "Post", active tabs, and important links.
- **Primary Text**: `#111827` (Near Black) — Used for post content, user names, and headings.
- **Secondary Text**: `#6B7280` (Medium Gray) — Used for timestamps, "@" handles, and placeholder text.
- **Action/Like Button**: `#E11D48` (Rose Red) — Used for "Like" or "Heart" interactions.

## 🌙 Dark Mode
Following modern UI guidelines (like Material Design), the dark theme does not use pure black. Instead, it uses dark grays to reduce eye strain and make shadows visible. Crucially, the accent and semantic colors are desaturated to pass contrast accessibility standards and prevent the colors from vibrating against the dark background.

- **Primary Background**: `#121212` (Deep Dark Gray) — Used for the main feed canvas.
- **Secondary Background**: `#1E1E1E` (Elevated Dark Gray) — Used for individual post cards, sidebars, and dropdown menus.
- **Primary Accent / Brand**: `#818CF8` (Soft Indigo) — A lighter, desaturated version of the light mode brand color to ensure it is legible and doesn't strain the eyes.
- **Primary Text**: `#F9FAFB` (Off-White) — Used for post content and user names. (Avoid pure white on dark backgrounds to reduce glare).
- **Secondary Text**: `#9CA3AF` (Muted Gray) — Used for timestamps and subtle UI elements.
- **Action/Like Button**: `#FB7185` (Soft Rose) — A brightened, pastel version of the like button for dark mode visibility.
